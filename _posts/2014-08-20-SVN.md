# 问题处理  
  * A: SVN	文件夹图标不显示  
    Q:

      ```
      HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\ShellIconOverlayIdentifiers
      ```

  * A: 锁定文件  
    Q:

      ```
      svn lock filename
      ```

  * A: 解除锁定  
    Q:

      ```
      svn unlock filename
      ```

  * A: svn log 显示修改文件信息
    Q:

      ```
      svn log -v
      ```

  * A: svn: E175013: Unable to connect to a repository at URL
    Q:

      ```
      1. 网络问题
      2.  右键点击本地副本，TortoiseSVN -> Settings -> Saved Data, 清除缓存
      ```
