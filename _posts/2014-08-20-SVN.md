# 问题处理  
  * SVN	文件夹图标不显示  

      ```
      HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\ShellIconOverlayIdentifiers
      ```

  * 锁定文件  

      ```
      svn lock filename
      ```

  * 解除锁定  

      ```
      svn unlock filename
      ```

  * svn log 显示修改文件信息

      ```
      svn log -v
      ```

  * svn: E175013: Unable to connect to a repository at URL

      ```
      1. 网络问题
      2.  右键点击本地副本，TortoiseSVN -> Settings -> Saved Data, 清除缓存
      ```
