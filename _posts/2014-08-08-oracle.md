#### Oracle异常处理

  * ORA-27101 shared memory realm does not exist  

```
    ORA-27101: shared memory realm does not exist	
    数据库未启动
    $ sqlplus /nolog
    SQL> connect sys/admin as sysdba
    SQL> startup
```

#### Oracle常用脚本
  * 创建表空间

```
--表空间
create tablespace A_DATA   
datafile 'C:\Oracle\oradata\ORACLE\A_DATA.DBF'
size 512M
extent management local ; 
create tablespace A_IDX   
datafile 'C:\Oracle\oradata\ORACLE\A_IDX.DBF'
size 256M
extent management local ; 
```
