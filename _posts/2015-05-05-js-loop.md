#### JS 遍历所有input并赋空值

```javascript
function resets()
{
  var controls = document.getElementsByTagName('input');
  for(var i=0; i<controls.length; i++){
    if(controls[i].type=='text'){
      controls[i].value='';
    }
  }
}
```
