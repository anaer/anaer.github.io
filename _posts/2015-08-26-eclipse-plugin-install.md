### Eclipse 插件安装方式
    1. Help -> Install New Software...
    2. 将插件下载后解压到[Eclipse_Home]/dropins目录下
    3. 将插件解压到自定义目录, 在[Eclipse_Home]/links目录下新增
      插件名.link
      path = [插件路径] #路径/需要替换为\
