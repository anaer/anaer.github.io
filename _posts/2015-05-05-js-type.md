#### js怎么判断一个对象是文本框(text)还是下拉框(select) 

```javascript
  var type = document.getElementById("tb_Test").type;
  if(type == "text"){
    alert("it is input box.");
  }
  if(type == "button"){
    alert("it is button.");
  } 
```
