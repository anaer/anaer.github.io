* 使用vimdiff来git diff
  使用 git d 文件名, 显示新老版本同步比较

```
git config --global diff.tool vimdiff
git config --global difftool.prompt false
git config --global alias.d difftool
```

* 在.rc中自定义比较函数
  一个文件中显示差异内容

```
function gd() {
    git diff --no-ext-diff -w "$@" | vim -R -
}
```
