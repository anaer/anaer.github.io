# 安装nodejs
  
  ```bash
  $ wget http://nodejs.org/dist/node-v0.4.5.tar.gz
  $ tar -xvf node-v0.4.5.tar.gz
  $ cd node-v0.4.5
  $ ./configure
  $ make install
  $ node -v
  ```
