#### SyntaxError: JSON.parse: unexpected character at line 1 column 1 of the JSON data
```js
JSON.parse(param); param为空或者不存在吧 转换前先判断
if(param){
    JSON.parse(param);
}
```

#### 防止window.open拦截的方式打开页面
```js
var tempwindow=window.open("_blank");
tempwindow.location="url";
```
