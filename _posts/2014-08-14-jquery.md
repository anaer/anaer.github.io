# JQuery插件编写  
	## jQuery 插件的结构

		```js
		//为避免冲突，将我们的方法用一个匿名方法包裹起来
		(function($) {

		    //扩展这个方法到jquery
		    $.fn.extend({

		        //插件名字
		        pluginname: function() {

		            //遍历匹配元素的集合
		            return this.each(function() {

		                //在这里编写相应代码进行处理

		            });
		        }
		    });

		 //传递jQuery到方法中，这样我们可以使用任何javascript中的变量来代替"$"
		})(jQuery);
		```
